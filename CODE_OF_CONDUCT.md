# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

We are committed to providing a welcoming and respectful community for everyone, regardless of background or experience level.

Expected behavior:

- Be respectful and considerate in communication
- Welcome differing viewpoints and constructive feedback
- Focus on what is best for the project and community
- Show empathy toward other community members

Unacceptable behavior includes any conduct that the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) defines as such.

## Scope

This Code of Conduct applies in all project spaces (issues, pull requests, discussions, code review) and when an individual represents the project in public.

## Enforcement

Reports of unacceptable behavior may be sent to the project maintainer at rajveer11vadnal@gmail.com. All reports will be reviewed and investigated promptly and fairly.

Maintainers who do not follow or enforce this Code of Conduct in good faith may face temporary or permanent repercussions as determined by other project leadership.
